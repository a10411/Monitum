Detailed Description

Esta classe implementa todas as funções que, por sua vez, implementam a parte lógica de cada request relativo aos comunicados Nesta classe, abstraímo-nos de rotas, autorizações, links, etc. que dizem respeito à API Porém, a API consome esta classe no sentido em que esta é responsável por transformar objetos vindos do DAL em responses. Esta classe é a última a lidar com objetos (models) e visa abstrair a API dos mesmos Gera uma response com um status code e dados
